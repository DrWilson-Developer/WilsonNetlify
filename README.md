# Netlify By Dr.Wilson

نکات مهم قبل از دیپلوی و استفاده :


۱-اول فایل رو از لینک زیر 👇(پایین صفحه) دانلود و سپس اون رو آنزیپ و بعدا فولدر رو آپلود کنید (به صورت زیپ آپلود نگردد چه در سیستم و چه در موبایل)


۲-هنگام ساخت کانفیگ در کانفیگساز ، ممکن است با ارور "دامنه شما خراب است" مواجه شوید ، نادیده بگیرید مهم نیست!


۳-از کانفیگ ها بصورت شخصی استفاده کنید ، در صورت استفاده همزمان چندین کاربر اکانت شما ساسپند خواهید شد.


۴-از اتصال شکن خود اطمینان حاصل کنید.


۵-بدون شک همه کانفیگ های ساخته شده پینگ نخواهند 
داشت پس معیار رو چند تا کانفیگ اولی قرار ندید و کلاینت (v2rayNg و ...) رو اسکرول کنید تا کانفیگ های دارای پینگ رو مشاهده کنید.
